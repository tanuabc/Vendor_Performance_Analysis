📊 Vendor Performance Analysis Dashboard

🚀 Project Overview

The Vendor Performance Analysis Dashboard is an interactive Power BI dashboard designed to analyze vendor performance using sales and procurement data. It provides actionable insights through KPIs, trend analysis, vendor comparisons, and brand-level performance, helping businesses make data-driven decisions.

---

🎯 Objectives

- Analyze vendor performance across multiple metrics.
- Compare vendors based on sales, volume, and profitability.
- Identify top-performing and underperforming vendors.
- Monitor brand-wise performance.
- Provide an interactive dashboard for business decision-making.

---

🛠️ Tools & Technologies

- Microsoft Power BI
- Microsoft Excel
- Power Query
- DAX (Data Analysis Expressions)

---

📂 Dataset

The dataset contains vendor transaction information, including:

- Vendor Number
- Vendor Name
- Brand
- Purchase Volume
- Sales Amount
- Profit
- Quantity
- Date
- Other business-related attributes

---

📈 Dashboard Features

Page 1 – Executive Summary

- KPI Cards
- Total Sales
- Total Profit
- Total Purchase Volume
- Total Vendors
- Sales Trend
- Top Vendors
- Brand Performance
- Interactive Filters

Page 2 – Vendor Deep Dive

- Vendor-wise Analysis
- Top 10 Vendors
- Bottom 10 Vendors
- Vendor Performance Comparison
- Scatter Plot Analysis
- Dynamic Slicers

---

📊 Key Insights

- Identified highest-performing vendors based on revenue and profit.
- Compared vendors using interactive visualizations.
- Analyzed brand contribution to overall sales.
- Highlighted opportunities for improving vendor performance.
